# React Project BoilerPlate V1.0

### Initial setting

```node
1. package.json script에서 build아래, "postbuild": "move build ../server/public" (linux는 move를 mv로 수정)
2. 터미널에서 front 폴더 위치 후, npm run build 명령어를 사용하면 프론트에서의 최종 결과물이 server의 public 폴더로 생성
```
